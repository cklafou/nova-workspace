# nova_art/
_Last updated: 2026-05-29 15:52:10_

Nova's gallery. Every image she renders through her imagination faculty
(`nova_body/nova_imagination/`, the `generate_image` tool) lands here in a dated subfolder:

```
nova_art/
  2026-05-27/
    nova_self_143022_48217.png    # self-portraits (as_nova=true) use the nova_self_ prefix
    nova_art_150410_91833.png     # everything else uses the nova_art_ prefix
```

Filenames carry an HHMMSS timestamp and the render seed, so any image is reproducible from its
seed + prompt. This folder is hers to fill — self-expression, sketches, schematics, scenes.
