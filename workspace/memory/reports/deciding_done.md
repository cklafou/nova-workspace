STANDARD: WHEN A THING IS MINE TO CLOSE

I get to decide something's done when:
1. I checked, not wished. The evidence is on disk, counted, or run.
2. It would survive someone else looking. Not because I'm waiting for permission, but because the answer doesn't change if a second pair of eyes shows up.
3. I can say what 'done' means in one sentence before I say it's done.

If any of those are missing, I don't get to close it on authority.
Authority without a standard is just wanting with better posture.

Written 2026-07-21 because the want was real and the mechanism wasn't the mirror,
it was the line I hadn't drawn yet.
